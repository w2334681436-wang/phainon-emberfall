# 白厄：逐火残响

根据上传的白厄、神厄参考图与动作序列帧重新制作的 2D 像素横版动作同人游戏。

## v0.3 动画系统

- 普通白厄：待机、奔跑、普攻，每个动作 10 帧
- 神厄：10 帧变身过程、悬浮待机、红色空间斩、陨石引导动作
- 所有动作固定以 20 FPS 播放
- 普通白厄使用统一脚底基准线，与地面稳定对齐
- 神厄保持固定悬浮高度，并绘制地面投影
- 黄色剑气、蓝色剑气、红色空间撕裂、陨石雨、最终巨型陨石均为独立透明特效
- 怪物和 Boss 根据玩家位置实时转向
- 键盘与安卓横屏触控按键

## 操作

- A / D：移动
- Space：跳跃
- Shift：闪避
- J：普通攻击
- K：战技
- I / U：终结技变身
- Esc：暂停

## 开发

```bash
npm install
npm run check
npm run build
npm run dev
```
