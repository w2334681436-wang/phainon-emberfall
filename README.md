# 白厄：逐火残响（Phainon: Emberfall）

原创像素风 2D 横版动作同人游戏原型。角色与世界观名称属于原权利方；本仓库不包含官方图片、音频或游戏文件，图形、粒子、透明序列动画和音效均由代码程序化生成。

## 当前首版已实现

- 三个连续关卡：灰烬前线、熔毁圣廊、焚风王座
- 三段普通攻击、二段跳、闪避无敌帧和连续组合技
- 战技、能量消耗及程序化技能特效
- 终结技：限时变身“神厄”，替换普通攻击与战技
- 近战怪、远程怪、飞行怪、精英怪
- 最终 Boss「绝灭大君·焚风」，包含三阶段攻击逻辑
- 回复、充能、攻击强化等关卡道具
- 生命、能量、终结技、连击和 Boss 血条
- GAL 式剧情头像、对白与字幕
- 程序化透明画布序列帧、粒子特效和 Web Audio 音效
- 键盘与手机触屏操作、自适应横屏画布
- localStorage 关卡进度存档、暂停和重试

## 操作

| 操作 | 键盘 |
|---|---|
| 移动 | A / D 或 ← / → |
| 跳跃 / 二段跳 | W / ↑ / Space |
| 闪避 | Shift |
| 普通攻击 | J |
| 战技 | K |
| 终结技 / 神厄变身 | L |
| 暂停 | Esc |

手机端使用屏幕下方的虚拟按键。

## 本地运行

需要 Node.js 20 或更高版本。

```bash
npm run check
npm run dev
```

浏览器打开 `http://localhost:4173`。

## Vercel 部署

1. 登录 Vercel，点击 **Add New → Project**。
2. 导入 `w2334681436-wang/phainon-emberfall`。
3. 保持仓库内 `vercel.json` 的默认配置，不需要手动填写框架。
4. 点击 **Deploy**。

Vercel 会运行 `npm run build`，校验完整游戏运行时并在 `dist/` 中生成可直接运行的普通 JavaScript 成品。以后每次 `main` 分支出现新提交，Vercel 都会自动重新部署。
